# zerohttp
http client written in like 8 hours for fun
## Features
- HTTPS support
- Basic and bearer authorization support
- Different methods support
- idk i forgor
## Installation and usage
You can just download zip archive or clone the repository, and then launch the `main.py` file
Or if you want to use it in your project you can just import the file from your .py file
You can also [download standalone binary](https://github.com/notzero4ik/zerohttp/releases) for arm64 and x86_64
